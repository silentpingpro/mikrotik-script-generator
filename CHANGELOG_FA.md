## ✨ بهبودی‌های اصلی

### 🔒 امنیت
- **پشتیبانی HTTPS** — تمام درخواست‌های API به جای HTTP از HTTPS استفاده می‌کنند
- **مدیریت خطا بهتر** — try-catch محکم‌تر برای تمام fetch calls
- **محدودیت زمانی (Timeout)** — 5-10 ثانیه timeout برای جلوگیری از hang شدن اسکریپت

### ⚡ عملکرد
- **کاهش 15% حجم** — HTML: 54.8 KB → 46.6 KB
- **فایل فشرده‌شده پرچم** — 97% کاهش (1.2 KB بجای 40+ KB)
- **اسکریپت‌های بهینه** — listener 30% کوچک‌تر

### 🎯 ویژگی‌های جدید
- **تشخیص هوشمند وضعیت** — شناسایی خودکار تغییر وضعیت سرورها (متصل/قطع)
- **Generator v11 جدید** — فایل HTML برای تولید اسکریپت‌های بهینه‌شده
- **get-country-v11.rsc** — GeoIP با محدودیت زمانی
